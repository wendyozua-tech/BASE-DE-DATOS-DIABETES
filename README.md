# BASE-DE-DATOS-DIABETES
La presente base de datos integra información correspondiente a un estudio observacional descriptivo realizado en una muestra de 271 participantes adultos, hombres y mujeres mayores de 18 años, residentes del estado de Sinaloa. La recolección de datos se efectuó durante el año 2025 con el propósito de analizar factores sociodemográficos, conductuales, antropométricos y ambientales asociados con condiciones de salud y riesgo metabólico.
La base de datos incluye variables sociodemográficas como edad, escolaridad y estado civil. Asimismo, se incorporaron mediciones antropométricas y clínicas, entre ellas peso, talla, circunferencia de cintura y hemoglobina glucosilada (HbA1c), esta última utilizada como indicador de control glucémico y riesgo metabólico.
Para la evaluación de factores psicosociales, de actividad física y condiciones del entorno alimentario, se aplicaron instrumentos validados y estandarizados. El nivel socioeconómico fue determinado mediante el cuestionario AMAI 2022. El estrés percibido fue evaluado a través de la Escala de Estrés Percibido PSS-14 (Perceived Stress Scale). La actividad física se midió utilizando el International Physical Activity Questionnaire (IPAQ). Además, se registró la distancia en metros a establecimientos alimentarios con el objetivo de evaluar la exposición a pantanos alimentarios y características del entorno alimentario.
La información fue obtenida mediante trabajo de campo realizado por la M.C.E. Wendy Paola Ramírez Ozúa y el Dr. Juan Carlos Guzmán Preciado durante el año 2025. La base de datos fue estructurada con fines académicos y de investigación científica, asegurando el tratamiento confidencial, anonimo y éticamente responsable de la información de los participantes.
